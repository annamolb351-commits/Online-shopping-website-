# Fashion Store - Online Shopping Website

This is a simple online shopping website project built using HTML, CSS, and JavaScript.  
It is designed like a mini Flipkart-style e-commerce website.

## 🛍 Features
- Display of dress products with name and price
- Add to cart functionality
- Cart item counter
- Auto delivery date display
- Simple and responsive UI design

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

## 📦 Project Type
Frontend Mini E-commerce Project (Student Project)

## 🎯 Purpose
To demonstrate basic e-commerce website functionality using front-end technologies.
